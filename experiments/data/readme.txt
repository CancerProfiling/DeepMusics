The pipeline for the preparation of the datasets
1. You can choose the omic types according to your own projects. It does not have to be the four omics type at the same time.
Some other clinical information including age,sex,weight or subtype,stage etc can also be included.
2. The omics data in one expriment have to be from the same cohorts.
3. The default set of omics data is snapped the gene level. 
If your data does not form the gene level. The pathway layer have to be redefined accordingly.
